# PDA Quest Editor
The quest editor for [Stalker PDA](https://pda.artux.net), written in Vite.js and TypeScript

To run the project, you need to install dependencies:
```shell
npm install
```
Launching a server for development:
```shell
npm run dev
```
Production build:
```shell
npm run build && npm run start 
```
